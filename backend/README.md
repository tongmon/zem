# zem-back-end  

Back-End project for ZEM (Zephly Express Manager)  
ZEM is a C++ based TCP server that helps manage the WMS.  

## Execution guide  

The ZEM server is structured under the Ubuntu environment.  
If you want to use this server, please build and run it on a Linux-based operating system.  

There are some installation steps you need to complete before executing ZEM, as outlined below.  

1. Move to the directory that contains the `build_env_setup.sh` file.  
1. Execute `chmod +x ./build_env_setup.sh` command.  
1. Build the project and bring the execution file from `Build` directory.  
1. Run the server with `nohup ./ZEM_v1.0.0.exe &` command.
